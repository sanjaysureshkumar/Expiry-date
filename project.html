Project Name : Expiry Date Notifier

This web application allows you to append the products to your account along with its Manufacturing Date.
After the Manufacturing Date and product is added, the web application calculates the current date and expiry date and notifies the user about the expiry date at regular intervals.Once the product is expired,the web app advises the user not yo use the product further.
